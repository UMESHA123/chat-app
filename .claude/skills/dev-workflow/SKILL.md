# Development Workflow

## Git

- Use feature branches
- Follow commit conventions:
  - feat:
  - fix:
  - refactor:
  - chore:

## Code Review

- Review for:
  - readability
  - performance
  - security

## Code Quality

- Enforce ESLint + Prettier
- Use TypeScript strictly

## Documentation

- Document APIs
- Add comments for complex logic
