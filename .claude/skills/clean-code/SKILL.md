# Clean Code Principles

## Naming

- Use meaningful variable names
- Avoid abbreviations

## Functions

- Keep functions small
- One responsibility per function

## Structure

- Avoid deep nesting
- Use early returns

## Comments

- Explain WHY, not WHAT

## Anti-Patterns

- ❌ Magic numbers
- ❌ Long functions
- ❌ Spaghetti logic
