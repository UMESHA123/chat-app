# Observability

## Pillars

- Logs
- Metrics
- Traces

## Logging

- Log:
  - errors
  - important events
- Use structured logs

## Metrics

- Track:
  - response time
  - error rate
  - active connections

## Tools

- Sentry (errors)
- Prometheus/Grafana (metrics)

## Anti-Patterns

- ❌ Console.log only
- ❌ No production visibility
