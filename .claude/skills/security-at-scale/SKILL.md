# Security at Scale

## Authentication

- Validate tokens on every request
- Expire tokens properly

## Authorization

- Ensure user belongs to chat before access

## Data Protection

- Encrypt sensitive data
- Use HTTPS everywhere

## Abuse Prevention

- Rate limiting
- Spam detection (basic rules)

## Anti-Patterns

- ❌ Trusting client roles
- ❌ Missing authorization checks
