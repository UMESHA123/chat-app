# Deployment & Reliability

## Principles

- Zero-downtime deployments
- Backward compatibility

## Techniques

- Blue-green deployment
- Rolling updates

## Database

- Avoid breaking schema changes
- Use migrations safely

## Recovery

- Have rollback plan
- Backup critical data

## Anti-Patterns

- ❌ Deploying without testing
- ❌ Breaking APIs
